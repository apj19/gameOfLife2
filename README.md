# Game Of Life 
Building game of life clone from [Conway’s Game of Life](https://playgameoflife.com/)

Thea main challenge was building the infinite grid, built grid virtualization using viewport logic. 
Zustand for state management. added Speed control tab.
and for Ui used shadcn.

## Tech Stack
1. React
2. React konva-for canvas grid drawing
3. Zustand state management
4. Shadcn- for Ui
